<h1>Simoni: sequencia de luzes</h1>
<p>Inspirado no jogo de memoria Simon says</p>
<p>Este projeto consiste na criação de um pequeno jogo utilizando um Arduino Nano, quatro botões, um buzzer, 3 LEDs e dois protoboards.
A ideia é servir como base para experimentos de eletrônica, jogos simples, lógica com estados, entrada e saída digital</p>
